ZetaPush Celtia

ZetaPush Celtia is the new version of the ZetaPush platform. The main features are the provision of more high-level services and the possibility to write your business logic on backend side with JavaScript/TypeScript. The strength of ZetaPush Celtia is that the deployment is easy. With some steps you have a deployed application.

Using ZetaPush Celtia help you to create your application in less time with more simplicity and comfort. There are already many provided cloud services :

  • Management of users

  • Chat

  • Upload of files

  • Data aggregation

  • …​

When you create your business logic with ZetaPush Celtia, you create a Custom Cloud Service. It will be deployed on the ZetaPush platform and provisionned for your application, like a standard Cloud Service.

You have 3 ways to create an application with ZetaPush Celtia :

  • From scratch, you create manually each file. Doc

  • With the CLI, you generate a squeleton of your application. Doc

  • With the Wizard on console.zetapush.com that help you to select what do you need. Doc
