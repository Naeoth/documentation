A Cloud Service is a set of Cloud Functions link:#cloud_function(see below) provided by ZetaPush. For example we a many Cloud Services like _Users management, real-time chat, data aggregation, …​

Cloud Function

To do a specific action with ZetaPush Celtia, we use a Cloud Function. For example, we can create an user with createUser(), from the users management cloud service.

Custom Cloud Service

A Custom Cloud Service is a Cloud Service that you have created. If you need to add some business logic in your application, you can create a Custom Cloud Service that regroup many Cloud Functions. After the deloyment, the Custom Cloud Services are callable like Cloud Service.
