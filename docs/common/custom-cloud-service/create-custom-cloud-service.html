1 If you want to use an existing cloud service you need to import it from @zetapush/platform
2 You inject the cloud service into your custom cloud service ("Stack" type)
3 Cloud function to add an item in a stack with the associated timestamp
4 You case use the async/await syntax in your custom cloud services
5 Cloud function to return all items from the stack

Dependencies

If you use existing cloud services provided by ZetaPush, you need to import them from @zetapush/platform. So you need to install this dependency with :

$ npm install --save @zetapush/platform

class DateCustomService { (1)
    constructor() {}

    formatDate(date) {
        return date.toLocalDateString("fr-FR");
    }


}

class HelloWorldService {
    constructor(dateCustomService) { (2)
        this.dateService = dateCustomService;
    }

    helloWorldWithDate() {
        const date = this.dateService.formateDate(new Date()); (3)
        return `Hello world at ${date}`;
    }
}
1 First custom cloud service only to manipulate dates
2 dateCustomService is an instance of DateCustomService
3 We use the DateCustomService in our HelloWorldService
