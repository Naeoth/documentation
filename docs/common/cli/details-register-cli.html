You can use the command zeta register in different contexts :

Create an anonymous account

To create a new anonymous you just need to launch :

Register your application
$ zeta register

It will create a new anonymous account on the ZetaPush platform with a new application and fill the .zetarc with the new informations.

You can also specify where you want to store your .zetarc file adding an argument on your command. For example, if you want to store your .zetarc file in the folder utils you can launch :

Register your application
$ zeta register ./utils

Create an activated account

This feature is work in progress

You can automatically create a new activated account. For this you can launch :


  
1


  
$ zeta register --email user@gmail.com

When you use --email, you specify that you want to create a new activated account.

A prompt ask you to create a password for your account. Then an email is sent to confirm your account. You have 7 days for it.

If you have an anonymous account in the .zetarc, the CLI ask you if you want to move your applications in the new accounts.
If you have an anonymous account in the .zetarc and you want to keep your data with the new account, you need to activate your account before the validity of anonymous account expires.
