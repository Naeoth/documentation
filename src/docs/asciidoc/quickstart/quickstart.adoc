= Quickstart

We explain each process to start development with ZetaPush using differents tools.

== How to start with the wizard

The wizard help you to select each cloud service you need and put them together. After that, you download a zip with your generated application.

== How to start with the CLI

With the CLI, you can create skeleton for your application (front and/or back).

== How to start from scratch

Here you create manually each file. You only use the documentation to create your application.
