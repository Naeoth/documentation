.Basic custom cloud service
[source, javascript, linenums]
----
class HelloWorldAsCustomCloudService {

    static get parameters() { <1>
        return [];
    }

    constructor() {} <2>
    
    helloWorld() { <3>
        return "Hello World";
    }
}
----
<1> You need to add static method named `parameters` that return the other services used (Only if you expose this service)
<2> You always need to add a constructor in a _custom cloud service_
<3> `helloWorld` is your first _cloud function_ in your _custom cloud service_ named *HelloWorldAsCustomCloudService*
