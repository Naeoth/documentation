.package.json
[source,json,linenums]
----
{
    "name": "myApp", <1>
    "main": "worker/index.js", <2>
    "dependencies": {
        "@zetapush/platform": "^0.11.1", <3>
        "@zetapush/cli": "^0.11.1" <4>
    }
}
----
<1> Name of your application. This is the same name that will be registered on your ZetaPush account.
<2> Entry point of your link:#_worker[worker]. This is in this file that you will expose your __custom cloud services__
<3> Dependency of the _ZetaPush Platform_ to write some _custom cloud services_ and access to existing _cloud services_ exposed by ZetaPush
<4> Dependency to use the link:#_command_line_interface_cli[CLI] (Command Line Interface). It will be useful to run your worker or deploy your code.
