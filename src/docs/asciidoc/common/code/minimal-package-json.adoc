<1> Entry point of your link:#_worker[worker]. This is in this file that you will expose your __custom cloud services__
<2> Dependency of the _ZetaPush Platform_ to write some _custom cloud services_ and access to existing _cloud services_ exposed by ZetaPush
<3> Dependency to use the link:#_command_line_interface_cli[CLI] (Command Line Interface). It will be useful to run your worker or deploy your code.
