<1> If you want to use an existing _cloud service_ you need to import it from `@zetapush/platform`
<2> You need to import `Inject` to import others _services_
<3> You specify that you use a `Stack` service
<4> You inject the _cloud service_ into your _custom cloud service_ ("Stack" type)
<5> _Cloud function_ to add an item in a stack with the associated timestamp
<6> You case use the _async/await_ syntax in your _custom cloud services_
