You can either upgrade your npm version or use npx utility.

[role=tab-container]
npm 5-

[role=tab]
Upgrade npm

.Upgrade npm
[source, console]
----
$ npm install -g npm@latest
----

.Use npm init
[source, console, subs="attributes+"]
----
$ npm init @zetapush {appName}
----

[role=tab]
Use _npx_ to launch the script

.Launch with npx
[source, console, subs="attributes+"]
----
$ npx @zetapush/create {appName}
----
