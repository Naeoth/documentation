<1> I launch `npm init @zetapush myApp` without parameters
<2> You type your email and a password to create your ZetaPush account
<3> An application is created in your workspace
<4> The _.zetarc_ file is like :
[source, json]
----
{
    "developerLogin": "user@gmail.com",
    "developerPassword": "password",
    "appName": "nameOfAppAutoGenerated"
}
----

TIP: You need to validate your ZetaPush account to run or deploy your application.
