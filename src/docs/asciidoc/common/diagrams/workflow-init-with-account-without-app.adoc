<1> I launch `npm init @zetapush myApp` with the _developer-login_ and _developer-password_ properties
<2> An application is created in your workspace
<3> The _.zetarc_ file is like :
[source, json]
----
{
    "developerLogin": "user@gmail.com",
    "developerPassword": "password",
    "appName": ""
}
----

TIP: If the _appName_ property in the _.zetarc_ is empty or missing, ZetaPush deploy or run the code on a new application.
