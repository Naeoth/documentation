.Deploy application (front and custom cloud services)
[source, console, subs="attributes+"]
----
$ npm run deploy
----

[TIP]
====
`npm run deploy` is a script alias defined in the `package.json` file.
The executed command under the hood is:
[source, console, subs="attributes+"]
----
$ zeta push
----
====
