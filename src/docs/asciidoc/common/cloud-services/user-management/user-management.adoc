== Definition

The _user management cloud service_ helps you to manage your users in your application.
This service provides many methods to create an account, connect a user into the application, reset the password, search a user...

The user management rely on the _UserManagementWorkflow_.
There are different workflows for the user management. The exposed methods are always the same but the process is different.

Depending of your use case, you can choose the proper workflow or the combine many workflows to create your own.
