Creation of a mini project like "Angular Getting started". This tutorial shows how to create a project with ZetaPush Celtia and how to use it.
For this, we will start from the CLI to create our project and we will deploy the application on the ZetaPush platform.

= Steps

* Init the project with the CLI
* Front development
* Interaction with ZetaPush services
* Debug of the application
* Deploy the application
* Create a custom service
* Testing
* Deploy the custom service
* Use the custom service
* Exploitation of the application
* Scalability
