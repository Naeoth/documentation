In this first step, you will initialize your *avengersChat* application. To do this, you only need to launch this command in your workspace :

[source, console]
----
$ npm init @zetapush avengersChat <1>
----
<1> _avengersChat_ is the name of your application

WARNING: If you using npm < 6, you need to launch `npx @zetapush/create avengersChat`
